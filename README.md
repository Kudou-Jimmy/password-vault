# Password Vault - 密碼管理程式

幫你把所有密碼安全地存在一個地方。只需要記住一組「主密碼」，就能管理所有網站、銀行、信用卡的帳號密碼。

你的密碼**只存在你自己的電腦和瀏覽器裡**，不會上傳到任何伺服器。

---

## 使用說明

### 第一步：開啟程式

在瀏覽器打開程式的網址，你會看到這個畫面：

```
🔐 建立密碼金庫
設定一組主密碼來保護你的資料

[________________]  ← 輸入你想設定的主密碼
[________________]  ← 再輸入一次確認

[   建立金庫   ]
```

- **主密碼**就像一把鑰匙，之後每次打開程式都要輸入它
- 至少要 8 個字，建議用一句你容易記住但別人猜不到的話
- 例如：`我家小黑2019年來的!`

> **請務必牢記你的主密碼！忘記的話，裡面的資料就再也打不開了。**

---

### 第二步：存入你的密碼

進入主畫面後，點右上角的「**+ 新增**」按鈕，選擇你要存什麼：

#### 🔑 一般網站帳號（Facebook、Google、蝦皮...）

選「**登入**」，填入：
- **標題** → 例如「Facebook」
- **網站** → 例如 `facebook.com`
- **帳號** → 你的帳號
- **密碼** → 你的密碼
- 填完按「**儲存**」

#### 🏦 網路銀行

選「**網路銀行**」，填入：
- **標題** → 例如「中國信託網銀」
- **銀行名稱** → 中國信託
- **身分證字號** → 你的身分證字號
- **使用者代號** → 銀行給你的代號
- **使用者密碼** → 你的網銀密碼

#### 🏧 金融卡（提款卡）

選「**金融卡**」，填入：
- **標題** → 例如「郵局金融卡」
- **銀行名稱** → 郵局
- **提款密碼** → ATM 提款的密碼
- **無卡提款密碼** → 如果有設定的話
- **磁條密碼** → 出國領錢用的密碼

#### 💳 信用卡

選「**信用卡**」，填入：
- **標題** → 例如「玉山 U Bear 卡」
- **卡號** → 信用卡卡號
- **持卡人** → 卡片上的英文名字
- **到期日** → 例如 `12/27`
- **CVV** → 卡片背面 3 碼數字
- **預借現金密碼** → 如果有設定的話

---

### 第三步：查看和複製密碼

需要用到密碼的時候：

1. 在左邊的列表中，**點一下**你要找的項目
2. 右邊會顯示詳細資料
3. 密碼預設會顯示成 `••••••••`（小圓點），這是為了保護你
4. 點「**顯示**」→ 就能看到密碼
5. 點「**複製**」→ 密碼會複製起來，到其他地方按「貼上」就好
6. **不用擔心**：複製的密碼會在 30 秒後自動從剪貼簿清掉

---

### 搜尋密碼

密碼太多找不到？在列表上方的搜尋框打幾個字就好：

```
🔍 [中國信託___]
```

打「中國」就會找出所有標題或內容有「中國」的項目。

---

### 用資料夾整理

密碼越來越多的時候，可以用資料夾分類：

1. 在左邊欄位的「資料夾」旁邊，點「**+**」
2. 輸入名稱，例如「銀行」「社群」「購物」
3. 新增或編輯密碼時，選擇要放在哪個資料夾

---

### 幫你產生安全的密碼

很多人的密碼太簡單容易被破解。程式可以幫你產生很強的密碼：

1. 新增或編輯密碼時，點「**🎲 密碼產生器**」
2. 會自動產生一組像這樣的密碼：`k8#Qm!pL2xR$nW9v`
3. 可以調整密碼長度（拉桿左右拉）
4. 滿意的話，點「**使用**」就會自動填入密碼欄位

也可以選「**助記詞組**」模式，產生像這樣比較好記的密碼：
```
brave-ocean-tiger-flame
```

---

### 檢查你的密碼安不安全

點列表上方的「**📊**」圖示，程式會幫你檢查：

- 哪些密碼太弱，容易被破解
- 哪些密碼重複使用（很危險！一個被盜，全部都會被盜）
- 哪些密碼太久沒換

---

### 備份你的密碼（很重要！）

如果換電腦或重灌瀏覽器，密碼會不見。所以要定期備份：

1. 點列表上方的「**📤**」圖示
2. 選「**🔒 加密匯出**」
3. 會下載一個檔案，把這個檔案存到安全的地方（例如 USB 隨身碟、iCloud、Google 雲端硬碟）

之後在新電腦要用的時候：
1. 打開程式，點下方的「**匯入既有金庫**」
2. 選擇你備份的那個檔案
3. 輸入你的主密碼，就能恢復所有密碼

#### 明碼匯出（進階）

如果你想把密碼匯出成一般的表格檔案（可以用 Excel 開）：
1. 點「**⚠️ 明碼匯出**」
2. 選 CSV 或 JSON 格式
3. **注意：這個檔案裡的密碼是可以直接看到的，千萬不要傳給別人或放在不安全的地方！**

---

### 刪錯了怎麼辦？

不用擔心！刪掉的東西會先放到「回收桶」：

1. 點左邊欄位的「**🗑️ 回收桶**」
2. 找到你誤刪的項目
3. 點「**恢復**」就回來了

如果確定不要了，點「**永久刪除**」就會真的刪掉。

---

### 離開或暫時不用

- **手動鎖定**：點左邊欄位底部的「**🔒 鎖定金庫**」
- **自動鎖定**：如果你 5 分鐘沒有操作，程式會自動鎖起來，要重新輸入主密碼才能打開
- 這樣就算你忘記關掉，別人也看不到你的密碼

---

### 切換深色模式

覺得螢幕太亮？點左邊欄位底部的「**🌙 深色模式**」，畫面會變成深色背景。再點一次「**☀️ 淺色模式**」就會變回來。

## 常見問題

**Q：忘記主密碼怎麼辦？**
A：很遺憾，無法找回。這是為了安全設計的——如果有人拿到你的電腦，他們也無法在不知道主密碼的情況下看到你的密碼。建議把主密碼寫在紙上，放在家裡安全的地方。

**Q：換電腦或換瀏覽器，密碼還在嗎？**
A：不會自動帶過去。請先用「📤 加密匯出」備份，再到新電腦「匯入既有金庫」。

**Q：手機可以用嗎？**
A：可以。用手機的瀏覽器打開同一個網址就能用，畫面會自動適應手機螢幕。

**Q：安全嗎？**
A：你的密碼用軍事等級的加密技術保護（AES-256-GCM），只有知道主密碼的人才能解開。而且所有資料都存在你自己的瀏覽器裡，不會傳到任何外部伺服器。

**Q：需要付費嗎？**
A：完全免費。

```bash
npm install
npm run dev
```

### 專案結構

```
src/
├── core/               # 核心模組（加密、事件系統、金庫邏輯）
│   ├── crypto.ts       # AES-256-GCM 加解密
│   ├── events.ts       # 事件總線（擴充用）
│   └── vault.ts        # 金庫 CRUD 操作
├── types/              # TypeScript 型別定義
├── contexts/           # React Context（金庫狀態、主題）
├── features/           # 功能模組
│   └── generator/      # 密碼產生器
└── ui/
    ├── components/     # UI 元件
    └── layouts/        # 頁面佈局
```

### 安全設計

- **零知識架構** — 伺服器僅提供靜態檔案，不處理任何密碼資料
- **加密金鑰** — 由主密碼透過 PBKDF2（600,000 次迭代）衍生
- **加密方式** — AES-256-GCM 提供加密與完整性驗證
- **本地儲存** — 所有資料加密後存於瀏覽器 localStorage
- **動態欄位** — 條目使用動態欄位設計，方便擴充
- **金庫版本化** — 檔案格式包含版本號，未來可向後相容地升級
- **事件系統** — 核心操作發出事件，功能模組訂閱，方便擴充新功能

### 部署

本專案為純靜態檔案，可部署至任何靜態託管服務：

- Cloudflare Pages
- Vercel / Netlify
