# Password Vault - 本地密碼管理程式

一個純前端、零後端的密碼管理工具。所有資料使用 AES-256-GCM 加密後儲存在瀏覽器本地，伺服器不接觸任何密碼資料。

## 功能

### 核心
- **AES-256-GCM + PBKDF2 加密** — 主密碼衍生加密金鑰，60 萬次迭代
- **多種條目類型** — 登入、網路銀行、金融卡、信用卡、身份、筆記，支援自訂欄位
- **資料夾 & 標籤** — 靈活分類管理

### 安全
- **密碼強度檢查** — 基於 [zxcvbn](https://github.com/dropbox/zxcvbn) 演算法
- **金庫健康報告** — 偵測弱密碼、重複密碼、過期密碼
- **自動鎖定** — 閒置 5 分鐘自動鎖定
- **剪貼簿自動清除** — 複製密碼後 30 秒自動清除

### 便利
- **密碼產生器** — 隨機密碼 & 助記詞組，可自訂長度與字元類型
- **模糊搜尋** — 即時搜尋標題、欄位、標籤
- **密碼歷史** — 自動記錄每次修改
- **回收桶** — 誤刪可恢復
- **深色 / 淺色模式**
- **RWD 響應式設計** — 支援手機使用

### 匯出入
- **加密匯出** — 備份或跨裝置轉移
- **明碼匯出** — JSON / CSV 格式，匯出前需確認主密碼
- **匯入** — 匯入加密金庫檔案

## 使用說明

### 首次使用

1. 啟動應用後，畫面會顯示「建立密碼金庫」
2. 設定一組**主密碼**（至少 8 個字元），這是解鎖所有資料的唯一鑰匙
3. 再次輸入確認後，點擊「建立金庫」即可進入主畫面

> **重要：主密碼無法找回。** 如果忘記主密碼，將無法存取金庫內的任何資料。請務必牢記。

### 新增密碼

1. 點擊右上角「**+ 新增**」按鈕
2. 選擇條目類型：

   | 類型 | 圖示 | 適用場景 | 預設欄位 |
   |------|------|----------|----------|
   | 登入 | 🔑 | 一般網站帳號 | 網站、帳號、密碼 |
   | 網路銀行 | 🏦 | 台灣網路銀行 | 銀行名稱、身分證字號、使用者代號、使用者密碼 |
   | 金融卡 | 🏧 | 提款卡 / 晶片金融卡 | 銀行名稱、卡號、提款密碼、無卡提款密碼、磁條密碼 |
   | 信用卡 | 💳 | 信用卡資訊 | 卡號、持卡人、到期日、CVV、預借現金密碼 |
   | 身份 | 👤 | 個人資料 | 姓名、電子郵件、電話、地址 |
   | 筆記 | 📝 | 自由文字 | 加密筆記內容 |

3. 填寫標題和各欄位，也可以新增自訂欄位
4. 選擇資料夾和標籤進行分類
5. 點擊「儲存」

### 使用密碼產生器

1. 新增或編輯「登入」或「網路銀行」條目時，點擊「🎲 密碼產生器」
2. 選擇模式：
   - **隨機密碼** — 自訂長度（8-64）、大小寫、數字、符號
   - **助記詞組** — 如 `brave-ocean-tiger-flame`，好記且安全
3. 點擊「重新產生」直到滿意，然後點「使用」套用到密碼欄位

### 搜尋

在條目列表上方的搜尋框輸入關鍵字，即時模糊搜尋：
- 條目標題
- 非敏感欄位內容（如網站名稱、帳號）
- 標籤名稱

### 整理分類

- **資料夾**：在左側欄點「+」建立資料夾，拖放或編輯條目時指定資料夾
- **標籤**：編輯條目時輸入標籤（逗號分隔），如 `工作, 社群`
- **我的最愛**：在條目詳情頁點「☆ 加入最愛」，方便快速存取

### 檢視與複製密碼

1. 點擊條目進入詳情頁
2. 敏感欄位預設顯示為 `••••••••`
3. 點「顯示」查看明碼，點「複製」複製到剪貼簿
4. 複製後 **30 秒自動清除剪貼簿**

### 密碼歷史

每次編輯條目時，系統自動記錄修改前的欄位值。在條目詳情頁底部點「顯示歷史紀錄」即可查看所有版本。

### 回收桶

- 刪除的條目會進入回收桶，不會立即消失
- 在左側欄點「🗑️ 回收桶」查看已刪除的項目
- 可選擇「恢復」或「永久刪除」

### 金庫健康報告

點擊條目列表上方的「📊」圖示，查看：
- 弱密碼清單
- 重複使用的密碼
- 超過 90 天未更新的密碼
- 整體密碼強度分布

### 匯出與備份

點擊條目列表上方的「📤」圖示：

- **🔒 加密匯出** — 匯出加密的 `.json` 檔案，可用於：
  - 定期備份
  - 轉移到其他電腦或瀏覽器
  - 透過雲端硬碟（iCloud / Google Drive）跨裝置同步
- **⚠️ 明碼匯出** — 匯出未加密的 JSON 或 CSV 檔案，所有密碼以明碼顯示。適用於：
  - 匯入其他密碼管理工具
  - 列印紙本備份

### 匯入金庫

1. 在鎖定畫面點擊「匯入既有金庫」
2. 選擇之前匯出的加密 `.json` 檔案
3. 輸入該金庫的主密碼即可解鎖

### 自動鎖定

閒置 5 分鐘後，金庫會自動鎖定，需重新輸入主密碼。也可以隨時點擊左側欄底部的「🔒 鎖定金庫」手動鎖定。

### 深色模式

點擊左側欄底部的「🌙 深色模式」或「☀️ 淺色模式」切換。系統會記住你的選擇。

## 安全設計

- **零知識架構** — 伺服器僅提供靜態檔案，不處理任何密碼資料
- **加密金鑰** — 由主密碼透過 PBKDF2（600,000 次迭代）衍生
- **加密方式** — AES-256-GCM 提供加密與完整性驗證
- **本地儲存** — 所有資料加密後存於瀏覽器 localStorage
- **動態欄位** — 條目使用動態欄位設計，不寫死 schema
- **金庫版本化** — 檔案格式包含版本號，未來可向後相容地升級
